## How to start

### 写新文章

``` bash
$ hexo new "My New Post"
```
<!-- more -->
### 启动服务

``` bash
$ hexo server || hexo s
```

### 编译文章

``` bash
$ hexo generate || hexo g
```

### 构建文章

```bash
$ hexo deploy || hexo d
```
